# Forgotten Runes Community Short Story Compendium

## Purpose

Demonstrate that **the community** can responsibly deliver a monetisable creative product under real constraints.

This project prioritises execution over perfection. The objective is a **Minimum Viable Product (MVP)** that proves coordination, quality control, follow-through, and fair handling of revenue (if any).

---

## Objective (Summary)

- Compile **10 original short stories** written by FRWC community members  
- Each story centres on **one primary FRWC token**  
- Set within the **Runiverse**  
- Published as a single short-story compendium  

Target completion: **14 April 2026**

---

## Timeline

- Submissions open immediately  
- Initial submission window: **30 days**  
- Extension window: **+30 days** (conditional)  
- Absolute submission limit: **60 days**

---

## Scope (High-Level)

- Target contributors: 10  
- Short fiction only  
- One primary FRWC token per story  
- Set in the Runiverse  

**Exclusions**
- Poetry  
- Serialised fiction  
- Lore encyclopaedia entries  
- Fanfiction outside the Runiverse

---
